docker安装mongo

发表于: 2020-05-15 09:48:28 | 字数统计: 129 | 阅读时长预计: 1分钟

1.查找镜像

docker search mongo

2.拉取镜像

docker pull mongo

3.查看是否拉取成功

[root@mail easy-mock-dev]# docker images
REPOSITORY          TAG                 IMAGE ID            CREATED             SIZE
mongo               latest              2b2cc1f48aed        7 days ago          388MB

4.创建挂载目录

mkdir -p /root/docker/mongo/db

5.通过镜像创建容器

docker run -p 27017:27017 -v /root/docker/mongo/db:/data/db --name mongo -d mongo

6.查看是否创建成功

[root@mail easy-mock-dev]# docker ps
CONTAINER ID        IMAGE               COMMAND                  CREATED             STATUS              PORTS                      NAMES
b67bab61a5aa        mongo               "docker-entrypoint.s…"   14 minutes ago      Up 14 minutes       0.0.0.0:27017->27017/tcp   mongo
